# datascience_project
DATA SCIENCE PROJECT

Il datset 'access to electricity' contiene informazioni riguardanti la percentuale di popolazione, nei diversi paesi, con accesso all'elettricitità, in un arco temporale di cinque anni. L'obiettivo del dataset è monitorare i progressi in relazione all'obiettivo di sviluppo sostenibile n.7 delle nazioni unite.
L'indicatore pricipale è la percentuale di persone che hanno accesso all'energia elettrica.
